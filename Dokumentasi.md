#PROSES SLR
RUMPUN ILMU = TEKNIK INFORMATIKA DENGAN PEMINATAN SOFTWARE ENGINEERING / WEB DEVELOPMENT
DETAIL = MENGEMBANGKAN HALAMAN WELCOME PAGE LARAVEL DENGAN TAILWINDCSS DAN VITE
P(Platform/System/Users) = WEB APP UNTUK FRONTEND DENGAN LARAVEL DAN VITE
I(Technology Intervention) = PHP, Laravel, TailwindCSS, Vite
C(Alternate Technology/Baseline) = Blade Template Biasa / HTML Standar
O(Outcome/Performance Matriks) = Tampilan Landing Page yang Responsif dan Interaktif, Integrasi Light/Dark Mode
S(Study Design/Evaluation Method) = User testing sederhana untuk melihat kemudahan penggunaan dan tampilan responsive

#PROSES RQ
RQ 1: Intervensi & Implementasi Teknologi (The "How" Question)
Bagaimana implementasi Laravel dengan TailwindCSS dan Vite dapat menciptakan halaman welcome yang responsif, modern, dan mudah dikembangkan?

RQ 2: Visualisasi & UX (The "So What" Question)
Bentuk layout dan interaktivitas seperti apa yang paling menarik dan memudahkan pengguna untuk memahami dan menavigasi halaman landing ini?

Penjelasan:
RQ ini berfokus pada outcome dari halaman welcome. Setelah halaman di-deploy, bagaimana pengalaman pengguna (UX) dapat optimal, termasuk penggunaan dark mode, tombol interaktif, hero section, dan visualisasi konten utama. Peneliti dapat melakukan user testing untuk menilai kemudahan navigasi dan estetika halaman.

#IMPLEMENTASI WEBSITE
1. cd /root/boilerplate
2. ./start.sh welcome-page-laravel
3. cd /root/perkuliahan/uts-pemweb
4. cd src
5. code .
6. Template website menggunakan TailwindCSS + Dark Mode + Hero Section
7. cd resources/views
8. mkdir layouts && mkdir pages && mkdir partials
9. cd ../../routes/
